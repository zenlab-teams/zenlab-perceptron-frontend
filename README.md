# ZenLab Perceptron — Frontend

Frontend aplikasi prediksi kelulusan mahasiswa. Dibangun dengan React + Vite.

## Tech Stack

- **React 19** + **Vite**
- **Tailwind CSS 4**
- **Recharts** — visualisasi data & klasifikasi
- **Oxlint** — linting

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Environment

Salin `.env.example` ke `.env` dan isi konfigurasi:

```
VITE_API_URL=http://localhost:8000
```

## Struktur

```
src/
├── components/    # Komponen UI
├── pages/         # Halaman
└── assets/        # Aset statis
```
